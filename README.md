## Sobre

O aplicativo foi feito em React-Native + Expo, com os dados em Redux e persistidos em Redux-Persist.

## Instalação

Para instalar é simples:

```
> Utilize yarn ou npm.
> yarn install
> yarn start
```

Lembre-se que é necessário também de ter um emulador Android ou IOS.
